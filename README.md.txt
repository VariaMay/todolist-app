    To-Do List Project

A simple To-Do List in pure JavaScript.

    Features

- Adding tasks
- Marking tasks as completed
- Deleting tasks
- Saving to `localStorage`
- Filtering tasks by: All / Active / Completed
- Animations when adding and completing tasks
- Responsive design

    How to run

1. Clone the repository:

https://github.com/VariaMay/todolist-app.git

2. Open the todo.html file in your browser.
